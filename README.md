# 酒店管理系统 java782

## 项目概述


## 技术栈

- 后端：Spring Boot
- 前端：Vue + ElementUI
- 运行环境：Node.js

## 功能模块

### 系统角色

- **管理员**
  - 公告信息管理
  - 酒店简介管理
  - 客房类型管理
  - 客房信息管理
  - 用户信息管理
  - 入住安排管理
  - 预定信息管理

- **用户**
  - 客房信息浏览
  - 公告信息浏览
  - 客房类型信息浏览
  - 预定客房
  - 预定信息管理
  - 个人资料管理
  - 在线评价

- **员工**
  - 客房预定信息管理
  - 用户入住安排管理

## 运行环境

- JDK 1.8
- IntelliJ IDEA
- MySQL
- Visual Studio Code

## 部署步骤

1. 安装并配置JDK 1.8环境。
2. 安装MySQL数据库，并导入数据库脚本。
3. 使用IntelliJ IDEA打开项目，配置相关数据库连接。
4. 使用VS Code打开前端项目，运行安装依赖命令。
5. 启动后端服务，然后启动前端项目。

## 目录结构

```
hotel_management_system/
├── backend/              # 后端代码目录
├── frontend/             # 前端代码目录
└── sql/                  # 数据库脚本目录
```

## 核心亮点

- **前后端不分离**：系统采用前后端不分离的架构，易于维护和扩展。
- **角色权限分明**：清晰的权限管理，确保各个角色的操作安全和效率。
